## 🎯 들어가며

"AI 비서에게 내 데이터를 맡기는 게 불안하다..."

그래서 떠오르는 게 **로컬 AI**입니다. 자신의 컴퓨터에서 돌아가는 AI는 **데이터가 외부로 나가지 않아서** privacy 걱정이 없습니다.

이 글에서는 비전공자도 할 수 있는 **로컬 AI 비서 구축 방법**을 소개합니다.

---

## 1️⃣ 로컬 AI란? 왜 써야 하나?

### 온라인 AI vs 로컬 AI

| 구분 | 온라인 AI (ChatGPT 등) | 로컬 AI (내 컴퓨터) |
|------|------------------------|---------------------|
| **데이터** | 서버로 전송됨 | 내 PC에만 존재 |
| **비용** | 유료 구독 필요 | 무료 (하드웨어만) |
| **속도** | 네트워크 의존 | 인터넷 없이 동작 |
| **성능** | 최상위 모델 | 중급 모델 수준 |

### 이런 분에게 추천
- 🛡️ **隐私 걱정**이 있는 분 (업무 자료,敏感情报)
- 💸 **비용 부담**을 줄이고 싶은 분
- ✈️ **인터넷 없이는 일 못 하는** 분 (항공사, 해외 출장)
- 🧪 **AI를 실험**해보고 싶은 분

---

## 2️⃣ 필요한 것

### 하드웨어
- **Mac**: Apple Silicon (M1/M2/M3) — RAM 16GB 이상 권장
- **Windows**: NVIDIA GPU — VRAM 8GB 이상 권장
- **리눅스**: NVIDIA GPU 또는 고성능 CPU

### 소프트웨어
- **Ollama** (가장 쉬운 방법) — https://ollama.com
- 또는 **LM Studio**, **Jan** 등

---

## 3️⃣ 5분 만에 시작하기 (Ollama)

### Step 1: Ollama 설치
```
https://ollama.com/download
```

 Mac: `brew install ollama`
 Windows: 설치 파일 다운로드 후 실행

### Step 2: 모델 다운로드
터미널 (또는 명령프롬프트)에서:

```bash
# 가장popular한 모델 설치 (약 4GB)
ollama pull llama3

# 가벼운 모델 (약 1GB, 느린 컴퓨터OK)
ollama pull mistral

# 中文対応 모델
ollama pull qwen
```

### Step 3: 실행
```bash
ollama run llama3
```

화면에 채팅창이 뜨면 성공! 💬

---

## 4️⃣ 로컬 AI로 할 수 있는 것

### ✅ 잘 되는 것
- **글쓰기 브레인스토밍**: 외부 전송 없이 아이디어 구상
- **코드 작성/리뷰**: 업무 코드를 외부에 보내지 않고 분석
- **문서 요약**:長い 문서를 오프라인에서 처리
- **번역**: 인터넷 없이도 동작하는 번역

### ⚠️ 제한적인 것
- 최신 정보 검색 (로컬 모델은 학습 데이터만 사용)
- 매우 복잡한 추론 (상위 클라우드 모델 대비)
- 이미지/파일 분석 (모델에 따라 불가)

---

## 5️⃣ 그래픽 UI가 필요하면? (Ollama + Chatbot UI)

터미널이 낯설다면:

```bash
# Chatbot UI 설치 (Node.js 필요)
npx chatbot-ui
```

또는 **LM Studio** 앱을 사용하면 **GUI로** 모델 관리 + 채팅이 가능합니다.
