## 🎯 OpenClaw란?

**OpenClaw**는 AI 비서를 내 컴퓨터에서 직접 운영할 수 있게 해주는 도구예요.

### 왜 보통 사람들은 어려워할까?

| 어려움 | 현실 |
|--------|------|
| ❌ 코드 작성 | OpenClaw는 **말로 명령**하면 됩니다 |
| ❌ 터미널/명령어 | 기본적으로 **채팅 방식**으로 동작 |
| ❌ 복잡한 설정 | **마법처럼** 알아서 해줌 |

---

## 📱 먼저 준비할 것

### 필수 아이템

1. **Mac 또는 Mac mini** (macOS 13 이상)
2. **텔레그램 계정** (무료)
3. **Google 계정** (드라이브/옵시디언 연동용)
4. **AI 구독** (GPT-5, Claude Pro 등 — 이미 있다면 OK)

---

## 🚀 5단계 간단 세팅

### 1단계: 설치 (5분)

OpenClaw를 다운로드하고 설치해요:

```bash
brew install openclaw
```

> 💡 Homebrew가 없다면? → [brew.sh](https://brew.sh)에서 먼저 설치

### 2단계: 기본 설정 (3분)

터미널에 입력:

```bash
openclaw setup
```

화면에 나오는 지시에 따라:
- Google 계정 연결
- 텔레그램 봇 생성
- 작업 폴더 지정

### 3단계: 텔레그램 연결 (2분)

1. 텔레그램에서 **@BotFather** 검색
2. `/newbot` 입력
3. 봇 이름 설정 (예: "나의 AI 비서")
4. **API 토큰** 복사
5. OpenClaw에 토큰 붙여넣기

### 4단계: Google 연동 (선택)

다음 명령으로 Google 서비스 연결:

```bash
openclaw configure
```

가능한 것들:
- 📧 Gmail 읽기/보내기
- 📅 Google Calendar 확인
- 📁 Google Drive 파일 관리
- 📝 옵시디언 문서 정리

### 5단계: 시작! (바로)

```bash
openclaw chat
```

이제 **채팅하듯이** 명령하세요:

> "오늘 일정 알려줘"
> "이 글 요약해줘"
> "릴스 영상 아이디어 떠올려줘"

## 💬 자주 하는 질문

### Q: 개발자 아니어도 쓸 수 있어요?

**A:** 네! 평범한 채팅처럼 사용할 수 있어요.

### Q: 비용이 얼마나 들어요?

**A:** OpenClaw 자체는 무료. AI 모델 비용만 별도로 (기존 구독 활용)

### Q: 어디서 쓸 수 있어요?

**A:** Mac이 켜져 있으면 어디서든 텔레그램으로 접근 가능

### Q: 영어를 못 해도 돼요?

**A:** 물론! 한국어로 자연스럽게 말하면 돼요
