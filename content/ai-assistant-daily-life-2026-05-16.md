## 🎯 들어가며

AI 비서(ChatGPT, Claude 등)를 그냥 "질문-답변" 도구로만 사용하고 계신가요?

其实 AI 비서는 **일상 문제를 해결하는 강력한 도구**입니다.

이 글에서는 실제 일상에 적용할 수 있는 **10가지 활용법**을 소개합니다.

---

## 1️⃣ 이메일 작성 – 시간 절약

### Before (직접 작성)
```
주kehr@naver.com에게:
안녕하세요, 다음과 같은 일정 조율이 필요합니다...
```

### After (AI 활용)
```
"친근하지만 격식 있는 한국어 이메일로:
-对方: 교수님
-内容: 논문 심사 결과 조stown
-期望: 이해와 양해 요청"
```

**결과:** 3분 소요 → 30초 ⏱️

---

## 2️⃣ 요리 레시피 – 냉장고 정리

오늘 냉장고에 남은 재료로 뭐 만들지?

```
"냉장고에: 닭고기, 양파, 당근, 커리 가루
가능한 레시피 3가지 추천해줘"
```

AI이 즉석에서 레시피를 만들어줍니다!

---

## 3️⃣ 여행 계획 – 맞춤 코스

```
"가족 4명(부모님 포함)과 3박 4일 서울 여행
- 부모님이 편하게 갈 수 있는 곳
- 아이들도 즐거운 곳
- 맛집 추천도 포함"
```

**출력:** 일별 추천 코스 + 지도 + 예약 팁

---

## 4️⃣ 회의록 요약 – 5분 → 30초

회의가 길어졌다면?

1. 음성 녹음 → 문자 변환
2. AI에 붙여넣기:

```
"아래 회의록을 3줄 요약해줘:
[회의록 내용 붙여넣기]"
```

---

## 5️⃣ 코드 작성 – 비개발자도 가능!

"코드 같은 건 못 하지만..."

```
"Google Sheets에서 A열 합계 구하는公式 알려줘"
```

**결과:** 바로 사용할 수 있는 수식

---

## 6️⃣ 여행 영문 메시지 – 해외여행

```
"호텔에 체크인 지각한다고 영어로 메시지 보내줘
- 30분 늦어짐
- 죄송함 표현
-的房间还能 stay 확인"
```

---

## 7️⃣ 선물 추천 – 어떤 걸까?

```
"동기 부부 결혼기념일 선물로 $100 이하
- 특별한 경험
- 집에서 쓸 수 있는 것
- 감성적인 것"
```

---

## 8️⃣ 공부 계획 – 자격증 시험

```
"컴퓨터활용능력 2급 시험 2주 후
- 하루당 공부 계획 세워줘
- 핵심考点 정리
- 모의고사 일정"
```

---

## 9️⃣ 유튜브 자막 → 블로그 글로

영문 유튜브 보면서 공부하고 싶다면:

1. 자막 복사
2. AI에 붙여넣기:

```
"이 유튜브 자막을 한국어 블로그 글로 변환해줘
- 제목: 
- 소제목: 
-본문: 
- 태그: "
```

## 💡 활용 팁

| 팁 | 설명 |
|------|------|
| **역할 지정** | "너는 experienced product manager야" 붙이면 더 좋은 답변 |
